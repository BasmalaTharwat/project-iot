# Smart Home Automation System
This project is a Smart Home Automation System developed using IoT technologies. It integrates multiple sensors, including a fire sensor with a buzzer, a rain sensor, an ultrasonic sensor, and a DHT sensor for temperature and humidity monitoring. The system also includes two servo motors to automate the opening and closing of a door and a window. A Flutter mobile application is used to interface with the hardware components, enabling real-time control and monitoring. Communication between the app and the IoT devices is handled through the MQTT protocol, allowing the system to send and receive notifications efficiently. This project aims to enhance home safety and convenience by automating responses to environmental changes and providing remote access to critical home functions.
